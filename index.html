FuelUX Core The nuts and bolts of FuelUX

Globalize

It enables complex culture-aware number and date parsing and formatting, including the raw culture information for hundreds of different languages and countries, as well as an extensible system for localization.

RequireJS

RequireJS is a JavaScript file and module loader. It is optimized for in-browser use, but it can be used in other JavaScript environments, like Rhino and Node. Using a modular script loader like RequireJS will improve the speed and quality of your code.

Handlebars

Handlebars provides the power necessary to let you build semantic templates effectively with no frustration. Mustache templates are compatible with Handlebars, so you can take a Mustache template, import it into Handlebars, and start taking advantage of the extra Handlebars features.

FuelUX Controls Easy to use controls with minimal code

Buttons

Access to simple styled buttons via input, button, or even link elements. The button library contains various sizes, colors, and states. Adding icons to buttons is a snap.

Modal Dialogs

Modal windows, also referred to as dialog windows, are very useful in web applications and can easily be applied. They have great benefits such as alerts, accepting form information from a user, images, and more.

FuelUX Components Custom tools that work with data

Editor

The FuelUX Editor component is an innovative tool that bridges a typical WYSIWYG editor with the ability to drag and drop widgets/gears (such as video, images, forms, polls, etc) to build ExactTarget emails and landing pages.

File Upload

The File Upload component is a unique tool that allows you to select a file or drag and drop multiple files to easily upload them.

Campaign Selector

Campaign selector is a custom component built so ExactTarget customers can easily manage their campaign associations for their emails.
